# How to run RAG Codebase Assistant

## 1) Go to project
```bash
cd "/Users/sairammaruri/Documents/New git projects/codebase-rag-assistant"
```

## 2) Install dependencies and CLI
```bash
pip install -r requirements.txt
pip install -e .
```

## 3) Set OpenAI API key
```bash
export OPENAI_API_KEY="sk-your-api-key-here"
```

## 4) Verify CLI is available
```bash
rag-code --help
```

## 5) Index a codebase
```bash
# local folder
rag-code index ./my-project

# or GitHub repo
rag-code index https://github.com/pallets/flask
```

## 6) Ask questions
```bash
# interactive chat
rag-code chat

# single-shot question
rag-code ask "How does authentication work?"
```

## 7) Useful extra commands
```bash
rag-code stats
rag-code impact src/auth.py
rag-code search "authentication" --limit 10
rag-code cache-info
rag-code cache-clear
```

## 8) Run API server (optional)
```bash
python -m src.api
```

Then call endpoints:
```bash
curl -X POST http://localhost:8000/index -H "Content-Type: application/json" -d '{"source":"./my-project"}'
curl -X POST http://localhost:8000/chat  -H "Content-Type: application/json" -d '{"query":"How does routing work?"}'
```

## Notes
- If `rag-code` is not found, run `pip install -e .` again.
- If parser modules are missing, run `pip install -r requirements.txt`.
